-> inicializa um repositorio git (utilizado apenas 1 vez)
git init

-> verificar o status do repositorio
git status

-> adicionar os novos arquivos no git para versionar
git add .

-> salva uma versao dos arquivos
git commit -m "motivo do qual estou salvando
"

-> envia os commits para o github
git push
